
# Bibliotech

## Status do Projeto
Em desenvolvimento

## Tecnologias utilizadas
- Java
- MySQL
- NetBeans

## Time de desenvolvedores
- Gabriel Marostica

## Objetivo do sistema
Sistema desktop para gerenciamento de biblioteca, permitindo o controle de alunos, livros, funcionários, empréstimos e devoluções.

## Funcionalidades
- Cadastro de alunos
- Cadastro de livros
- Cadastro de funcionários
- Registro de empréstimos
- Registro de devoluções
- Controle de estoque de livros
- Login de usuário
